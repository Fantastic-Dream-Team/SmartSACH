# Frontend SmartSACH-FDT

Vistas separadas del antiguo monolito PHP de `Pruebas XAMPP/php`.

- `login.html`: inicio de sesion.
- `registro.html`: creacion de cuenta.
- `dashboard.html`: panel del usuario.
- `assets/js/api.js`: configuracion de llamadas al backend.

En local puedes abrir los HTML directamente o entrar desde el backend en `http://localhost:10000`.
